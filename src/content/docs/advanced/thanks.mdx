## Open Source References

This theme uses/references/derives content from the following projects:

Framework:

- [astro](https://github.com/withastro/astro)
- [tailwincss](https://github.com/tailwindlabs/tailwindcss)

Content rendering:

- [mdx](https://github.com/mdx-js/mdx/)
- [tailwindcss-typography](https://github.com/tailwindlabs/tailwindcss-typography)
- [@unifiedjs/rehype](https://github.com/rehypejs/rehype)
- [@unifiedjs/remark](https://github.com/remarkjs/remark)

Integrations:

- Code block: [Shiki](https://github.com/shikijs/shiki)
- Image lightbox: [Medium Zoom](https://github.com/francoischalifour/medium-zoom)
- Comment: [Waline](https://github.com/walinejs/waline)
- Search: [Pagefind](https://github.com/cloudcannon/pagefind)

Others:

- Compress: [playform/compress](https://github.com/PlayForm/Compress)
- Image compress: [sharp](https://github.com/lovell/sharp)
